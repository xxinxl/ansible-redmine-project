Автоматизация развертывания Redmine с помощью Ansible и Docker
Обзор проекта
В рамках данной работы была реализована задача по автоматизации настройки серверной инфраструктуры и деплоя веб-приложения. В качестве целевого приложения использован образ Redmine, развернутый на облачной виртуальной машине.

Выполненные задачи:
Создание облачной инфраструктуры: На платформе Yandex Cloud была вручную развернута виртуальная машина под управлением Ubuntu 24.04 LTS.

Настройка сетевого доступа: Сконфигурированы группы безопасности для доступа по протоколам SSH (порт 22) и HTTP (порт 80).

Разработка конфигурации Ansible: Создан Inventory-файл для управления хостом и Playbook для автоматизации всех этапов настройки.

Автоматизация деплоя:

Реализована автоматическая установка движка Docker на удаленный сервер.

Настроен запуск контейнера Redmine с пробросом портов и политикой автоматического перезапуска.

Контроль версий: Весь процесс инфраструктурных изменений зафиксирован в Git-репозитории.

Технологический стек:
Облачный провайдер: Yandex Cloud.

ОС: Ubuntu 24.04.

Инструменты автоматизации: Ansible.

Контейнеризация: Docker.

Приложение: Redmine (official image).

Результат:
Приложение успешно развернуто и доступно в глобальной сети.
